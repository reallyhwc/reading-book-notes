# 《世界的逻辑》

- **作者**: 马兆远
- **出版**: 浙江科学技术出版社（湛庐文化）, 2025年9月
- **ISBN**: 9787573918949
- **阅读状态**: 阅读中

## 内容简介

以"如何用理性应对不确定性"为核心主线，从古希腊哲学贯穿到量子力学前沿。没有晦涩公式，用"科学史 + 思维进化"双重视角，还原理性与科学的本质：它们不是"终极答案"，而是人类应对未知的"思维工具"。
