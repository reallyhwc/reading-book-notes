# 《人工智能之不能》

- **作者**: 马兆远
- **出版**: 中信出版社, 2020年3月
- **ISBN**: 9787521713794
- **阅读状态**: 暂停中（读到第二章，已切换至《世界的逻辑》）

## 内容简介

马兆远（清华大学未来实验室首席研究员、牛津大学物理学博士）从哥德尔不完备定理出发，贯通数学、逻辑学、量子力学和物理学，在原理层面论证了人工智能的局限性——有些真理人类能够认识到，但不能为算法所描述。核心论点：不存在一个超级通用人工智能系统。
