# 读书笔记

以程序员视角阅读 AI / 科学 / 思想类书籍，记录技术概念解析、个人思考与延伸讨论。

笔记由 **AI 辅助产出**（纲要与摘要），**个人补充**（思考、质疑、联想）。每章按固定 A~E 板块组织，另有 raw/ 目录保存对话式深度讨论。

## 目录结构

```
books/<book-slug>/
├── README.md          # 书籍信息与目录索引
├── notes/             # 正式笔记（A~E 模板，结构化摘要）
│   ├── chapter-01.md
│   └── ...
├── raw/               # 对话记录（个人感想、专题讨论、质疑）
│   ├── chapter-01.md
│   ├── topic-*.md     # 跨章节专题
│   └── ...
└── media/             # 图片等资源
```

### notes/ vs raw/

| | notes/ | raw/ |
|---|---|---|
| **定位** | AI 辅助产出的结构化摘要 | 对话式个人笔记 |
| **风格** | A~E 板块，百科词条式 | 自然语气，保留论据与态度 |
| **读者** | 想快速了解书中要点的人 | 未来的自己 |

### 笔记板块 (A~E)

- **A. 章节概述** — 简要概括本章内容
- **B. 技术要点** — 技术概念解析，含背景知识和延伸
- **C. 人文/思想洞察** — 人文关怀、科学家精神、社会影响
- **D. 关键人物/事件** — 重要人物、里程碑事件
- **E. AI 补充背景** — 书中未展开但相关的背景知识

## 协作方式

通过 [Claude Code](https://claude.ai/code) 辅助阅读：对话中聊感想与质疑，AI 负责提炼并写入 raw/，积累后反向更新 notes/。详见 [CLAUDE.md](CLAUDE.md)。
